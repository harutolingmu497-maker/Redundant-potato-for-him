このフォルダには簡単な GUI ダイアログ（Web モーダル）のサンプルがあります。

ファイル:
- `web_modal.html` : シンプルなモーダルダイアログの静的デモ。ブラウザで開いて動作を確認できます。

使い方:
1. `examples/web_modal.html` をブラウザで開きます（ダブルクリックで開けます）。
2. 「削除ダイアログを開く」ボタンを押してダイアログを表示します。
3. Tab / Shift+Tab / Esc でキーボード操作を確認できます。

注意:
- 実運用ではフォーカストラップやスクリーンリーダー向けのラベル強化（aria-describedby など）を追加してください。
